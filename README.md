# 🌺 Awesome Gyaru Culture & Y2K Aesthetic

Welcome to the open-source archive dedicated to preserving and celebrating the bold, rebellious Japanese Gyaru subculture and early 2000s Y2K fashion.

## 📖 About This Project
This repository is a collaborative list of resources, history, and styling guides for anyone interested in alternative J-fashion. From the 1990s Kogal schoolgirl rebellion to the extreme hyper-tanning of Manba, we document it all.

✨ **For complete, in-depth styling guides, expert makeup tutorials, and rich visual archives, please visit our main platform: [Gyaru Online](https://gyaru.online/)** ✨

## 🎀 Core Sub-styles We Track
* **Kogal (コギャル):** The 90s school uniform pioneers.
* **Hime Gyaru (姫ギャル):** The ultimate princess aesthetic (Jesus Diamante, bouffant hair).
* **Manba / Yamanba (マンバ):** The extreme, iconic hyper-tanned look.
* **Neo Gyaru (ネオギャル):** The modern, western-influenced digital renaissance.
